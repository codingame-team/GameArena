# Avatar Images

Place PNG avatar images here (64x64 or 128x128 recommended).

## Required avatars:
- `no_avatar.png` - Default placeholder (gray circle with question mark)
- `my_bot.png` - User's bot avatar (robot icon)
- `boss.png` - Boss avatar (crown icon)
- For other bots: use bot name in lowercase with underscores (e.g., `random_walker.png`)

## Temporary SVG files
The `.svg` files in this directory are temporary placeholders.
Replace them with actual PNG images for better performance and appearance.
